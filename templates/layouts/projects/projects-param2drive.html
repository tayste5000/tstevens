Param2Drive

Click here to use param2drive
you will be prompted to authorize google drive access for this site before proceeding.

param2drive uses the Biopython library to compute some common chemical properties of a protein from a user-submitted sequence. This information is then formatted into a text document and inserted into the user's google drive service. This project is driven by the goal of finding an easier, more consistent, and more shareable way of storing common biochemical information than in lab notebooks.



If you notice any errors or want to suggest new changes let me know by opening an issue on the github repo. also welcome any constructive additions to the project on github.
